API ROUTES:
- GET /api/signals → returns all current signals as JSON
- GET /api/watchlist → returns all stocks with LTP, change, volume, signal
- GET /api/history → returns last 50 signals from SQLite DB
- GET /api/stock/ → returns data + signal for one stock
- GET /api/chart// → returns OHLCV data for charting

SIGNAL GENERATION (mock for now, CNN later):
- For each stock, generate a realistic mock signal
- Randomly assign BUY/SELL/HOLD with weighted probability (40% BUY, 30% SELL, 30% HOLD)
- Pattern names: randomly pick from ["Hammer", "Doji", "Bullish Engulfing", "Bearish Engulfing", "Morning Star", "Evening Star", "Shooting Star", "Three White Soldiers", "Dark Cloud Cover", "Piercing Line"]
- Entry = realistic mock price for each stock
- Target = entry * (1.008 for BUY, 0.992 for SELL)
- SL = entry * (0.996 for BUY, 1.004 for SELL)
- Confidence = random between 58-94%
- Cache signals for 60 seconds

SQLite DB:
- Save every generated signal to signals table
- Table: id, symbol, timeframe, signal_type, entry, target, sl, pattern, confidence, timestamp, status
