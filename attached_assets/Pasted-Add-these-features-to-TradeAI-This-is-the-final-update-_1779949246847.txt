Backend - paper trades table in SQLite:
CREATE TABLE paper_trades (
  id INTEGER PRIMARY KEY,
  symbol TEXT,
  signal_type TEXT,
  entry_price REAL,
  target_price REAL,
  sl_price REAL,
  quantity INTEGER,
  invested_amount REAL,
  pattern TEXT,
  timeframe TEXT,
  confidence REAL,
  status TEXT DEFAULT 'Active',
  pnl REAL DEFAULT 0,
  entry_time TEXT,
  exit_time TEXT,
  exit_price REAL
)

Auto-update trade status every 60 seconds:
- Fetch current price for each active trade
- If current price >= target: status = "Target Hit", pnl = positive
- If current price <= stoploss: status = "SL Hit", pnl = negative
- Calculate pnl = (exit_price - entry_price) * quantity for BUY
- Calculate pnl = (entry_price - exit_price) * quantity for SELL
