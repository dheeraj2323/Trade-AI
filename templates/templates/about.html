TradeAI

CNN-Based Candlestick Pattern Recognition for Indian Equity Markets

Academic Project · CSE 2026
How CNN Pattern Recognition Works

TradeAI uses a Convolutional Neural Network (CNN) trained on over 10,000 candlestick chart images to detect technical patterns in real-time market data. The model converts OHLCV price data into visual chart representations and classifies them into 19 known candlestick patterns.

1️⃣
OHLCV Ingestion
Open, High, Low, Close, Volume data fetched for 6 timeframes (1m to 1W)
2️⃣
Chart Rendering
Last 20 candles rendered as 224×224px image using matplotlib with dark theme
3️⃣
CNN Classification
MobileNetV2 classifies pattern → BUY/SELL/HOLD signal with confidence score
Technology Stack
Python 3.11 Flask 🤖 TensorFlow 2.x 📱 MobileNetV2 SQLite 📊 Chart.js 🕯️ Lightweight Charts 🅱️ Bootstrap 5 📈 NSE Market Data
Model: MobileNetV2 pre-trained on ImageNet, fine-tuned on 10,000+ candlestick chart images across 19 pattern classes. Input: 224×224 RGB image. Output: pattern class + confidence score. Multi-timeframe consensus combines 6 independent CNN inferences.
